# Python Dev Toolbox 🛠️

A collection of lightweight Python scripts designed to automate daily development tasks and optimize workflow.

## 🚀 Featured Tool: Port Killer
Tired of the "Address already in use" error when running Flask or Django? This script identifies the process running on a specific port and terminates it instantly.

### How to Use
1. Clone the repository: `git clone https://github.com/kcnsoftware/python-dev-toolbox.git`
2. Navigate to `src` folder.
3. Run the script: `python port_killer.py`
4. Enter the port number (e.g., 5000) and press Enter.

### Technologies
- **Language:** Python 3.x
- **Modules:** `os`, `subprocess`, `signal`
- **Platform:** Cross-platform (Windows & Linux support)
